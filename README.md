# ColorPV
A command line color previewing tool

## Use
To preview a color, type it's hexadecimal value:
```sh
colorpv 00c864
```

## Examples
You can type colors in four different formats:

```sh
colorpv "#c86400"
```
```sh
colorpv 0064c8
```
```sh
colorpv b47
```
```sh
colorpv "#835"
```
